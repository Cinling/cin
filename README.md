# cin
一个 http 和 websocket 多模块的开发框架。创作主要灵感来源于 yii2 高级模板

# 特性

- 数据库版本管理（使用 go-migrate）
- 数据库ORM(使用xorm)
- 自动路由
- 应用模式启动

# 计划

    1.构建 http 和 websocket 自动路由

## 0.1.0

    2.添加 xorm 作为orm引擎
